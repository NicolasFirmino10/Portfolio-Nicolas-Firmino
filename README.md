# Portfólio Profissional - Nicolas Firmino

Um portfólio web moderno e responsivo desenvolvido para apresentar projetos, habilidades e experiências profissionais de forma elegante e interativa.

## Sobre o Projeto

Este portfólio foi desenvolvido com foco em:
- **Design Responsivo**: Adaptável para desktop, tablet e mobile
- **Performance**: Carregamento rápido e otimizado
- **UX/UI Moderno**: Interface limpa e profissional
- **Acessibilidade**: Seguindo boas práticas de desenvolvimento web

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível
- **CSS3**: Estilização moderna com Flexbox e Grid
- **Font Awesome**: Ícones minimalistas e profissionais


### Seções Principais
1. **Hero Section**: Apresentação pessoal com foto e call-to-action
2. **Sobre Mim**: Valores e motivações profissionais
3. **Benefícios**: Diferenciais e competências
4. **Projetos**: Showcase de trabalhos realizados
5. **Depoimentos**: Recomendações de colegas
6. **Contato**: Call-to-action para networking
